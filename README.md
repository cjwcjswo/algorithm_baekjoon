# 백준 알고리즘 문제 풀이

이 저장소는 백준 온라인 저지(BOJ)의 알고리즘 문제들을 Python으로 풀이한 내용을 정리한 것입니다.

## 📅 문제 풀이 일지

### [2025년 11월 24일](./20251124/README.md)
기본 입출력, 사칙연산, 조건문, 반복문, 문자열 처리 등 기초 문제들을 풀이했습니다.

**주요 학습 내용:**
- 기본 입출력 및 사칙연산
- 조건문 (if-elif-else)
- 반복문 (for, while)
- 문자열 처리 및 포맷팅
- 빠른 입출력 (sys.stdin.readline)

### [2025년 11월 25일](./20251125/README.md)
리스트 조작, 문자열 처리, 집합 활용 등 중급 문제들을 풀이했습니다.

**주요 학습 내용:**
- 리스트 기본 연산 및 조작
- 리스트 슬라이싱 및 역순 처리
- 집합(Set) 활용
- 문자열 처리 및 변환
- 딕셔너리 활용
- 예외 처리

### [2025년 11월 26일](./20251126/README.md)
문자열 패턴 매칭, 빈도 분석, 팰린드롬 검사 등 고급 문자열 처리 문제들을 풀이했습니다.

**주요 학습 내용:**
- 문자열 패턴 매칭 및 KMP 알고리즘
- 문자열 빈도 분석
- 팰린드롬(Palindrome) 검사
- 반복문과 문자열 조작
- 리스트 연산 및 차이 계산

### [2025년 11월 27일](./20251127/README.md)
2차원 배열 처리, 딕셔너리 활용, 문자열 패턴 검사 등 중급 문제들을 풀이했습니다.

**주요 학습 내용:**
- 2차원 배열(행렬) 처리 및 연산
- 딕셔너리를 활용한 데이터 매핑
- 가중 평균 계산
- 그룹 단어 체커 (문자열 패턴 검사)

### [2025년 11월 28일](./20251128/README.md)
진법 변환, 그리디 알고리즘, 수학적 패턴 분석 등 다양한 알고리즘 문제들을 풀이했습니다.

**주요 학습 내용:**
- 진법 변환 (10진수 ↔ B진법)
- 그리디 알고리즘 (거스름돈 문제)
- 수학적 패턴 분석
- 2차원 격자 및 영역 계산

### [2025년 12월 1일](./20251201/README.md)
수학적 계산, 패턴 분석, 기하학적 문제 등 다양한 수학 문제들을 풀이했습니다.

**주요 학습 내용:**
- 수학적 계산 및 올림 처리
- 패턴 분석 및 수열 (분수찾기)
- 기하학적 패턴 분석 (벌집)
- 수학 함수 활용
